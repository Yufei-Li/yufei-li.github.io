1

javac hello.java // produce a java class program
java hello // java + the name of java class program

  • the java program must contain public static void main(String[] args) to run

  • if there is a public class in the program, the name of program must be same as name of class, so only one public class in one program

  • every class in program will generate a .class file, usually one source code file define one class
