BeautifulSoup将复杂的文档转换成一个树形结构,所有的对象可以归为四种类型：Tag, NavigableString, BeautifulSoup, Comment。对于爬虫来说，主要使用其中的遍历文档树和搜索文档树

  • 创建BeautifulSoup对象

#模拟爬虫已经爬完该网页，开始解析该html文档
file = open("./baidu.html", "rb")
html = file.read()

bs = BeautifulSoup(html, "html.parser")  #告诉BeautifulSoup解析的是html文件，使用html.parser来解析

BeautifulSoup的四种类型

Tag

即节点，比如title、head，tag类型主要有两个属性，name和attributes属性，可通过.name和.attrs方法获取属性值，其中.attrs返回字典类型

1
2
3

 print(bs.title)    #输出为第一个title标签及其所有内容:<title>百度一下，你就知道</title>
 print(type(bs.title))      #输出为<class 'bs4.element.Tag'>
 print(bs.a.attrs)    #输出为一个标签里的所有属性，以字典的形式保存

NavigableString

标签里的内容（字符串）

1
2
3

 print(bs.title.string)     #百度一下，你就知道
 print(type(bs.title.string))    #<class 'bs4.element.NavigableString'>

BeautifulSoup

表示整个文档，一种特殊的tag属性，支持遍历文档树和搜索文档树的大部分方法，但是没有name和attrs属性

1

 print(type(bs))      #<class 'bs4.BeautifulSoup'>

Comment

一个特殊的NavigableString，文档中的注释，输出时不包含注释

文档的遍历

遍历文档树，就是通过方法获取指定的节点和节点集，包括:方法、子节点、父节点、兄弟节点、前进后退等

方法

获得一个节点最简单的方法是用.方法，格式.<节点名>，比如：

1
2

 print(beautifulsoup.a)   
 #<a class="sister" href="http://example.com/elsie" id="link1">Elsie</a>

通过.方法可以获取文档中的第一个要搜索的节点名节点。

子节点

通过子节点内的方法获取其子节点或者子孙节点。

.comtents    获取其直接子节点，并返回一个列表

.children    获取其直接子节点，返回一个生成器

.descendants 获取其子，子的子…的生成器

1
2
3
4
5
6
7
8
9

 print(beautifulsoup.body.contents)      #直接输出一个列表，每一元素都是一个子节点，里面包含一些"\n"

 for i in beautifulsoup.body.children:    #children返回一个生成器，遍历得到直接子节点

        print(i)

 for i in beautifulsoup.body.descendants:   #获取其所有的子孙节点，获取一个直接子，然后是其下面的子，子的子...

        print(i)

父节点

通过父节点，获取父和父辈的节点和节点集。

.parent：某一个节点的父节点

.parents：某一个节点的可遍历父节点集

1
2
3
4
5
6
7

 print(beautifulsoup.a.parent)   #结果:p标签包裹的三个a标签

 print(beautifulsoup.a.parents)  #<generator object PageElement.parents at 0x00000265C2DC3408>

 for i in beautifulsoup.a.parents :#返回可遍历所有的父辈节点的生成器

    print(i)

兄弟节点

通过兄弟节点，获取同一个父标签下的前一个兄弟和后一个兄弟的节点和节点集

.next_sibling 获取下一个兄弟节点

.previous_sibling 获取前一个兄弟节点

.next_sibings 所有该节点的下一个兄弟节点的遍历对象

.previous_siblings 该节点的所有前面兄弟节点的遍历对象

前进和后退

前进和后退是模拟解析器的解析过程，前进，下一个解析目标；后退就是前一个解析目标。该方法的结果和兄弟节点的方法的结果一般是不同的。

.next_element 下一个解析节点

.previous_element 前一个解析节点

.next_elements 后面的解析节点遍历对象

.previous_elements 往前的解析节点遍历对象

.string、.strings、.stripped_strings

.string 当标签下面只有一个字符串时，正常输出字符串内容，否则输出none

.strings 当tag下面有多个字符串时，会返回标签下面的所有字符串的生成器

.stripped_strings 返回当前标签下面的所有字符串（去除空格和空行）的生成器

文档的搜索

.find_all()/.find()

搜索文档树重要使用.find_all()和.find方法，下面以find_all方法为例，两者的区别在于find直接返回找到的第一个结果，如果没有结果的话是NONE，而find_all是返回符合条件的tag的list，如果没有结果的话输出空的list

过滤器的概念：字符串/列表/正则表达式/True/方法

find_all( name , attrs , recursive , string , **kwargs )

name：查找所有名字符合name的tag，参数可以是所有的过滤器，包括字符串/列表/正则表达式/True/方法；

attrs：指定的tag属性attrs={“id”:“link3”},传递一个字典进去

recursive：确定是否只搜索直接子节点，如果只想搜索直接子节点，=false

string：需要查找的字符串内容，可以是字符串/列表/正则/true

keyword：需要搜索的tag属性

limit：设定需要查找的数量

 #正则表达式搜索：使用search（）方法来匹配内容
     import re
     print(bs.find_all(re.compile("a")))
     
     #返回一个列表，所有含有字母a的标签都会被显示，比如head、meta等等，head及其子标签都储存在同一个元素中

 #方法：传入一个函数，根据函数的要求来搜索
     def name_is_exists(tag):
         return tag.has_attr("name")
     
     print(bs.find_all(name_is_exists))
     ```

 #kwargs 参数

      `bs.find_all(id="head")`返回id=head以及下面所有子标签

CSS选择器

在写 CSS 时，标签名不加任何修饰，类名前加点，id名前加 #

在这里我们也可以利用类似的方法来筛选元素，用到的方法是 soup.select()，返回类型是 list

  1. 通过标签名查找 
1
2

 print soup.select('title') 
 #[<title>The Dormouse's story</title>]
  1. 通过类名查找
1
2

 print soup.select('.sister')
 #[<a class="sister" href="http://example.com/elsie" id="link1"><!-- Elsie --></a>, <a class="sister" href="http://example.com/lacie" id="link2">Lacie</a>, <a class="sister" href="http://example.com/tillie" id="link3">Tillie</a>]
  1. 通过 id 名查找
1
2

 print soup.select('#link1')
 #[<a class="sister" href="http://example.com/elsie" id="link1"><!-- Elsie --></a>]
  1. 组合查找

组合查找即和写 class 文件时，标签名与类名、id名进行的组合原理是一样的，例如查找 p 标签中，id 等于 link1的内容，二者需要用空格分开

1
2

 print soup.select('p #link1')
 #[<a class="sister" href="http://example.com/elsie" id="link1"><!-- Elsie --></a>]
  1. 子标签查找
1
2

 print soup.select("head > title")
 #[<title>The Dormouse's story</title>]
  1. 属性查找

查找时还可以加入属性元素，属性需要用中括号括起来，注意属性和标签属于同一节点，所以中间不能加空格，否则会无法匹配到

1
2

 print soup.select('a[href="http://example.com/elsie"]')
 #[<a class="sister" href="http://example.com/elsie" id="link1"><!-- Elsie --></a>]

以上的 select 方法返回的结果都是列表形式，可以遍历形式输出，然后用 get_text() 方法来获取它的内容
