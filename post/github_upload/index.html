Github

2020-08-19
cs技能点

Contents

  1. 在github上建立相应的repository

  2. 在终端进入想要上传的文件夹，git init

  3. 配置 ssh , 输入：ssh-keygen -t rsa -C “yufeili_cloris@163.com” (登录github的邮箱)

  4. 一直输入回车直到ssh配置成功

  5. 复制ssh到剪切板，pbcopy < /Users/liyufei/.ssh/id_rsa.pub.

  6. 在github里，点击头像–>settings–>SSH and GPG keys–>new SSH key, 随便取一个title，将SSH key复制进去

  7. 在终端输入ssh -T git@github.com 查看是否配置成功

    • 如果出现 The authenticity of host ‘git.net (116.211.167.152)’ can’t be established， 就再执行命令就再执行命令：ssh-add /Users/liyufei/.ssh/id_rsa

  8. 设置github 登录名以及登陆邮箱

    git config –global user.name “your name”

    git config –global user.email “your_email@youremail.com”

  9. git status –> git add . –> git commit -m “随便写点备注”

  10. git remote add origin git@github.com:用户名/项目名.git ，后面的内容需要在github上找到项目相应的SSH

    • 如果报错，可以尝试删除远程仓库地址再重新添加，删除远程仓库地址 git remote rm origin

  11. git push -f origin master
