  • :
  • : adds an element to the head of a list
1
2
3

'A' : " BIG MESS" --"A BIG MESS"

54 : [45, 69, 27] --[54, 45, 69, 27]

  • compare ++ and :

    • ++ takes a pair of lists of the same type and : takes an element and then a list of that type of element

    • ++ can do both prepending and appending, but : can only do prepending

    • 1
2
3

      54 : [45, 69, 27] -- [54, 45, 69, 27]
[45, 69, 27] : 54 -- error
[45, 69, 27] ++ [54] --[45, 69, 27, 54]

  • !!

  • use the !! operator to get a list member by index number; indices start from 0

  • raise error if index is out of the range

1
2

[3.4, 7.89, 9.4, 12.0] !! 3 --12.0

tuples

compare lists and tuples:

    • lists only have homogeneous elements
    • tuples can have heterogeneous elements
    • lists have flexible size: grow, shrink
    • tuples have a fixed size
